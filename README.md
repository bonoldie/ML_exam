## Quickstart 

### Linux 

Create and activate the conda environment

```bash
conda env create -f environment.yml && conda activate $(head -1 environment.yml | cut -d':' -f 2)
``` 

Launch the notebook 

```bash
jupyter notebook
```

or use the notebook the vs-code extension
